集成 OpenAI/DeepSeek，智能优化文档质量和翻译

♿

无障碍文档

自动生成无障碍特性说明和平台合规文档

🚀 快速开始

pip install ai-docgen

# 分析项目
ai-docgen analyze ./my-project

# 生成文档
ai-docgen generate ./my-project --format md

# AI 增强
ai-docgen enhance ./docs/api.md

📦 安装

pip install ai-docgen

# 或从源码安装
git clone https://github.com/zizegak916-glitch/ai-docgen.git
cd ai-docgen
pip install -e .
